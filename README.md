
## Overview
This repository contains SQL queries written to solve common data analysis and interview-style problem statements using sample datasets.

The focus is on writing efficient, readable queries and understanding core SQL concepts used in real-world scenarios.

## Topics Covered
- SELECT, WHERE, GROUP BY
- Joins (INNER, LEFT, RIGHT)
- Window Functions
- Subqueries
- CASE Statements

## Use Case
These queries are created for:
- SQL interview preparation  
- Hands-on practice of analytical queries  
- Understanding query logic and data relationships  

The problems are designed to simulate real business and reporting scenarios.

## Tools
- MySQL  
- PostgreSQL  
*(Syntax may vary slightly between databases)*

## Notes
Each SQL file focuses on a specific concept and includes commented queries for better understanding.
